# pyhkconnect

## Install

```bash
pip install pyhkconnect
```

## APIs

### 获取北上沪港通持股数据

```python
import pyhkconnect as hkc
hkc.statistics_northbound_sh()
```

### 获取北上深港通持股数据

```python
import pyhkconnect as hkc
hkc.statistics_northbound_sz()
```
